# Generated History Artifacts

This directory contains historical generated artifacts from earlier milestone scripts.
They are kept for auditability only and are not current public release claims.

In particular, historical `A+`, `certified`, `complete`, or similar labels in this directory must not be interpreted as external certification, production readiness, or current project status.
Current release-facing status is defined by `README.md`, `TECHNICAL_REPORT.md`, `KNOWN_ISSUES.md`, and `docs/VALIDATION_STATUS.md`.
