# Russian Section Fragments

Use this directory for modular TeX fragments.

Suggested split:
- `core/` — problem, method, validated state
- `runtime/` — runtime story, experiments, negative results
- `wal/` — WAL framing, formal objects, roadmap
- `appendices/` — extra tables, protocol, artefact notes
- `templates/` — reusable TeX section templates

`main_ru.tex` should remain the single file that aggregates these parts.
