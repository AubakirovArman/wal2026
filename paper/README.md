# Paper Layout

This directory now has a modular paper structure.

Rules:
- keep new TeX fragments at 200 lines or fewer when practical
- use `main_ru.tex` as the master aggregator file
- write new sections in `sections_ru/`
- keep experiment-to-paper bridges in `experiments/`
- keep tables and figures in their own folders

Folders:
- `sections_ru/` — TeX fragments for the Russian paper
- `experiments/` — notes mapping experiments to paper claims
- `tables/` — standalone table sources or notes
- `figures/` — figure sources and captions
