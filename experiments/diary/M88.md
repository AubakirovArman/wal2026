# M88: KV-cache Integration Test

## Date
2026-04-23

## Goal
Test WALKVCache module end-to-end: encode, decode, DynamicCache conversion, lazy caching, and different budgets.

## Results

### Test 1: Basic Encode/Decode
- 4 layers encoded and decoded successfully
- Max decode error: K=0.106, V=0.066 — within tolerance
- Compression ratio: 2.46x (K=32/4, V=16/4)

### Test 2: DynamicCache Conversion
- `WALKVCache.to_dynamic_cache()` works with transformers >= 4.36
- Compatible with `model(input_ids, past_key_values=cache)`

### Test 3: Lazy Decode
- First access triggers decode
- Second access returns cached tensor (identical reference)
- `clear_decoded_cache()` frees memory

### Test 4: Budget Sweep
| Budget | K atoms/coeffs | V atoms/coeffs | MSE | Ratio |
|--------|---------------|----------------|-----|-------|
| High | 256/16 | 128/16 | 0.00000002 | 1.39x |
| Medium | 128/8 | 64/8 | 0.00000218 | 1.68x |
| Low | 64/8 | 32/4 | 0.00003474 | 2.00x |

Higher budgets = lower MSE but less compression. Low budget achieves 2× compression with negligible error.

## Production API

```python
from wal.v1.kv_cache import encode_kv_cache, decode_kv_cache, WALKVCache

# Encode
wal_cache = encode_kv_cache(past_key_values, k_budget=(256, 16), v_budget=(64, 8))

# Use with transformers
from transformers.cache_utils import DynamicCache
cache = wal_cache.to_dynamic_cache(device="cuda")
outputs = model(input_ids=input_ids, past_key_values=cache)

# Decode
decoded = decode_kv_cache(wal_cache, device="cuda")

# Stats
print(f"Compression: {wal_cache.compression_ratio:.2f}x")
print(f"Memory saved: {wal_cache.memory_saved_mb:.1f} MB")
```
